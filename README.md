# 📋 Attendance Tracker

A simple and interactive **Attendance Tracker** built using **HTML, CSS, and JavaScript**. This project allows users to record student attendance by marking each student as **Present** or **Absent**. The application automatically updates attendance statistics and also provides an option to delete individual student records.

This project was developed to strengthen frontend development skills by practicing DOM manipulation, event handling, and dynamic user interface updates.

---

## ✨ Features

- Add students by entering their names
- Mark students as **Present** or **Absent**
- Automatically update:
  - Total Students
  - Present Students
  - Absent Students
- Delete individual student records
- Instant updates without refreshing the page
- Simple, clean, and user-friendly interface
- Responsive design for different screen sizes

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Attendance-Tracker.git
```

2. Open the project folder.

3. Double-click **index.html** or open it using **Live Server** in Visual Studio Code.

4. Start adding student attendance.

---

## 💻 How to Use

1. Enter the student's name.
2. Click the **Present** button to mark attendance as Present.
3. Click the **Absent** button to mark attendance as Absent.
4. The attendance summary updates automatically.
5. Use the **Delete** button to remove any student record.
6. The total, present, and absent counts are updated instantly.
